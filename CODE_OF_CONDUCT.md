# Code of Conduct

This project follows a simple standard:

- be rigorous
- be respectful
- do not turn technical disagreement into personal friction

## Expected Behavior

- Bring evidence, not posturing.
- Critique code, claims, and design decisions directly and clearly.
- Respect privacy, especially because this project touches highly personal data.
- Keep discussions welcoming to new contributors, even when the technical bar is high.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks
- Publishing private data, chat histories, or generated psychological profiles
- Bad-faith review behavior, trolling, or deliberate derailment

## Enforcement

If a discussion or contribution crosses the line, maintainers may edit, remove, lock, or reject the related content.

For security or privacy-sensitive incidents, use the guidance in [SECURITY.md](SECURITY.md).
